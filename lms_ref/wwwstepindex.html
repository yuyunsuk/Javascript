과정소개

스프링 프레임워크는 Java를 기반으로 하는 소프트웨어 시장에서 가장 폭넓게 사용하는 오픈소스 프레임워크다. 하지만 초급 개발자가 스프링을 학습하고 숙달하기까지는 많은 시간과 노력이 필요하다. 스프링 학습을 어렵게 만드는 가장 큰 요인은 복잡한 라이브러리 의존성과 XML 설정에 있다. 이런 라이브러리 의존성과 XML 설정을 해결하면서 스프링에 대한 접근성을 획기적으로 개선시킨 것이 바로 스프링 부트다. 본 과정은 풍부한 실습을 바탕으로 스프링 부트의 동작 원리와 구조를 이해하는 것을 목적으로 한다.  

학습목표

- 스프링 부트의 특징과 동작 원리를 설명할 수 있다. 

- 스프링 부트가 제공하는 기능을 이용하여 간단한 웹 애플리케이션을 작성할 수 있다. 

- 스프링 부트로 작성한 웹 애플리케이션을 테스트할 수 있다. 

- 스프링 부트로 작성한 웹 애플리케이션의 로그를 제어하고 빌드할 수 있다. 

학습대상

? 관련 직군 및 직무 : 소프트웨어 개발

? 학습 대상자 : Java 기반의 소프트웨어를 개발하고 유지보수하려는 개발자

? 학습 난이도 : 중
